# Automação de Planilhas Excel - C# (ClosedXML)

## Descrição
Este projeto é um mini-sistema em C# que automatiza planilhas do Excel. Ele permite:
- Formatar relatórios para cobrança
- Preparar relatórios para análise
- Comparar dados entre relatórios (em desenvolvimento)

O objetivo é transformar tarefas repetitivas e manuais em um fluxo automatizado usando código C# e a biblioteca ClosedXML.

## Pré-requisitos
- Visual Studio ou VS Code com .NET SDK instalado
- Biblioteca ClosedXML (`Install-Package ClosedXML`)

## Como usar
1. Abra o projeto no Visual Studio ou VS Code.
2. Atualize os caminhos dos arquivos Excel no código (`caminhoArq`, `caminhoArq2`).
3. Execute o programa.
4. No menu, escolha a opção desejada:
   - (1) Relatório Formatado 
   - (2) Relatório para análise
   - (3) Comparação de Relatórios
   - (4) Sair do Programa

## Estrutura do Código
- `Program.cs`: Contém o menu principal e a lógica de manipulação das planilhas usando ClosedXML.

## Observações
- Este projeto é um exemplo de automação prática em C#.
- Pequenos ajustes nos caminhos de arquivos podem ser necessários.
